The zsc Toolchain

ZScript ships with zsc, a fast, cross-platform compiler and runner built on top of Bun. You don’t need extra tooling or complex setup — everything is included out of the box.

With zsc, you can compile your project ahead of time or compile and run it instantly during development: 

zsc build main.zs
zsc run main.zs

Under the hood, zsc transpiles ZScript to modern JavaScript and uses Bun’s high-performance runtime to execute it. You can even access the full Bun CLI directly through ZScript: 

zsc bun install
zsc bun add antlr4

This tight integration gives ZScript native access to the JavaScript ecosystem while keeping the language itself clean, structured, and focused on correctness.

About ZScript

ZScript is designed to feel simple on the surface while enabling powerful compile-time transformations underneath. Because it transpiles, ZScript can safely introduce features that are difficult or impossible to express in traditional runtime-only languages.

We are actively adding new keywords and language features such as queue, a unified loop construct, and more structured control-flow abstractions — all without sacrificing performance.
